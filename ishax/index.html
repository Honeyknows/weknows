Features

Detection Capabilities

Visibility across in-memory scripts, process creations, network connections, registry keys, and multi-tenant database management.

In-Memory AMSI ETW Interception

Native C# ETW watcher service (amsi_watcher.exe) listening on provider {2A576B87-09A7-520E-C21A-4942F0271D67} to record un-obfuscated script content before execution.

Dual-Layer pySigma Correlation

Merges AMSI memory regex patterns with pySigma YAML rules across a 30-second correlation window to trigger high-confidence alerts.

T1027 Entropy Analysis

Calculates Shannon Entropy and Base64 density on script blocks. High obfuscation scores trigger alert priority flags in the alert queue.

Multi-Tenant Database Isolation

Master directory database (master.db) routing operational events to isolated per-tenant SQLite databases (tenant_<uuid>.db).

Host Network Isolation

Active Response firewall script (isolate.ps1) to sever compromised host network traffic while maintaining agent connection ports.

AI Incident Triage

Assistant endpoints supporting OpenAI, Gemini, and Ollama models for root-cause analysis and remediation script recommendations.

MITRE ATT&CK Matrix

Technique Coverage

Monitors and flags post-exploitation techniques across the MITRE ATT&CK framework.

T1059.001
PowerShell Execution
Intercepts memory script blocks, -EncodedCommand, hidden windows, and web download cradles.
T1027
Obfuscated Files & Scripts
Evaluates Base64 density, XOR strings, character concatenations, and high Shannon entropy.
T1036
Process Masquerading
Identifies PE metadata vs. file execution path mismatches (e.g. svchost.exe outside System32).
T1219
Remote Access Tool Abuse
Detects unauthorized AnyDesk, TeamViewer, Ngrok, and Atera remote management utilities.
T1543.003
Windows Service Creation
Monitors Sysmon EID 1, System EID 7045, Security EID 4697, and sc.exe execution.
T1547.001
Registry Run Keys
Detects persistence writes to HKLM\...\Run and RunOnce via Sysmon EID 13.
T1003
OS Credential Dumping
Monitors LSASS process memory access attempts via comsvcs.dll minidumps and Mimikatz.
T1055
Process Injection
Tracks remote thread creation (Sysmon EID 8) and cross-process memory operations.
System Architecture

Pipeline & Data Flow

From endpoint kernel events to backend detection and process tree graph renderers.

PIPELINE DATA FLOW

1. Endpoint Agent

Sysmon64 + Native ETW AMSI Service

2. Log Forwarder

Wazuh Agent over Port 1514 (Encrypted TLS)

3. Ingestor Engine

Normalizes raw log stream into SQLite schema

4. Dual Detector

AMSI Regex + pySigma YAML 30s Correlation

5. Server & Dashboard

FastAPI Backend & React Process Ancestry UI

# 2. Check operational status of all services
